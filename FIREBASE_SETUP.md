# 🔥 Firebase Configuration Guide - Study Buddy

Panduan lengkap setup Firebase untuk push notifications & authentication.

---

## 📋 Prerequisites

- ✅ Firebase project (gratis)
- ✅ Google account
- ✅ Flutter project yang sudah ada
- ✅ Android emulator / physical device
- ✅ (Opsional) iOS device/Mac untuk iOS setup

---

## 🚀 Step-by-Step Configuration

### Step 1: Buat Firebase Project

1. Buka https://console.firebase.google.com
2. Klik **"Add project"** atau **"Create a project"**
3. Isi nama project: `study-buddy-[your-name]`
4. **Enable Google Analytics** (opsional, recommended)
5. Klik **"Create project"**
6. Tunggu sampai selesai → Klik **"Continue"**

---

### Step 2: Tambah Android App

1. Di Firebase Console, klik icon **Android** 🤖
2. Isi **Android package name**:
   ```
   com.example.studybuddy
   ```
   > ⚠️ Cek package name kamu di: `android/app/build.gradle` → `namespace` atau `applicationId`

3. (Opsional) Isi **App nickname**: `Study Buddy Android`
4. (Opsional) **Debug signing certificate**:
   ```bash
   # Windows - jalankan di terminal:
   keytool -list -v -keystore %USERPROFILE%\.android\debug.keystore -alias androiddebugkey -storepass android -keypass android
   ```
   Copy SHA-1 fingerprint ke Firebase

5. Klik **"Register app"**

---

### Step 3: Download `google-services.json`

1. Firebase akan generate file `google-services.json`
2. Klik **"Download google-services.json"**
3. **PENTING**: Letakkan file di:
   ```
   L:\StudyBuddy\android\app\google-services.json
   ```

4. Klik **"Next"** di Firebase Console

---

### Step 4: Tambah iOS App (Jika ada Mac/iOS device)

1. Klik icon **iOS** 🍎
2. Isi **iOS bundle ID**:
   ```
   com.example.studybuddy
   ```
   > Cek di: `ios/Runner.xcodeproj` → Bundle Identifier

3. Download `GoogleService-Info.plist`
4. Letakkan di:
   ```
   L:\StudyBuddy\ios\Runner\GoogleService-Info.plist
   ```

---

### Step 5: Install FlutterFire CLI

```bash
# 1. Install FlutterFire CLI global
dart pub global activate flutterfire_cli

# 2. Pastikan PATH sudah benar (Windows)
# Tambahkan ke PATH: %LOCALAPPDATA%\Pub\Cache\bin

# 3. Test installation
flutterfire --version
```

**Jika error "flutterfire not found":**

```powershell
# Windows PowerShell - tambahkan ke PATH
$env:PATH += ";$env:LOCALAPPDATA\Pub\Cache\bin"

# Atau edit Environment Variables secara permanent:
# 1. Win + R → sysdm.cpl
# 2. Advanced → Environment Variables
# 3. Edit PATH → Add: %LOCALAPPDATA%\Pub\Cache\bin
```

---

### Step 6: Generate Firebase Config

```bash
# Navigate ke project root
cd L:\StudyBuddy

# Login ke Firebase (akan buka browser)
firebase login

# Run FlutterFire configure
flutterfire configure \
  --project=study-buddy-[your-project-id] \
  --platforms=android,ios,web \
  --out=lib/firebase_options.dart
```

**Apa yang terjadi:**
- ✅ CLI akan detect Firebase project kamu
- ✅ Generate `lib/firebase_options.dart`
- ✅ Auto-configure Android & iOS
- ✅ Setup semua Firebase products yang aktif

**Output yang diharapkan:**
```
i Found 1 Firebase project.
✔ Select a Firebase project: study-buddy-xxx
✔ Which platforms should your configuration support? android, ios, web
i Firebase config file written to lib/firebase_options.dart
```

### Step 7: Enable Firebase Services

#### A. Firebase Authentication (Optional - untuk login)

1. Firebase Console → **Authentication** (menu kiri)
2. Klik **"Get started"**
3. Enable **Email/Password**:
   - Toggle ON
   - Klik **Save**

#### B. Firebase Cloud Messaging (FCM) - Untuk Push Notifications

1. Firebase Console → **Cloud Messaging** (menu kiri)
2. **Sudah otomatis aktif** saat project dibuat
3. Catat **Server Key** (untuk backend nanti):
   - Settings ⚙️ → Project settings → Cloud Messaging
   - Copy **Server key**

#### C. Firebase Analytics (Optional)

1. Firebase Console → **Analytics** → **Dashboard**
2. **Sudah otomatis aktif**
3. Bisa track user behavior

### Step 8: Update Flutter Code

#### 8.1 Uncomment Firebase Dependencies

File: `pubspec.yaml`

```yaml
dependencies:
  # Sudah ada, tidak perlu diubah:
  firebase_core: ^2.24.2
  firebase_messaging: ^14.7.9
  flutter_local_notifications: ^16.3.2
```

```bash
# Install dependencies
flutter pub get
```

#### 8.2 Initialize Firebase di `main.dart`

File: `lib/main.dart`

```dart
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'package:firebase_core/firebase_core.dart';
import 'firebase_options.dart'; // ← Generated by flutterfire configure

import 'core/theme/app_theme.dart';
import 'features/main_navigation/presentation/main_navigation_screen.dart';

void main() async {
  // Required: Initialize widgets binding before any async calls
  WidgetsFlutterBinding.ensureInitialized();

  // 🔥 Initialize Firebase
  await Firebase.initializeApp(
    options: DefaultFirebaseOptions.currentPlatform,
  );

  runApp(
    const ProviderScope(child: StudyBuddyApp()),
  );
}
```

#### 8.3 Update Notification Service

File: `lib/core/services/notification_service.dart`

Ganti placeholder dengan implementasi Firebase yang sebenarnya. Lihat template di bawah.

### Step 9: Android Configuration

#### 9.1 Update `android/build.gradle`

```gradle
buildscript {
    dependencies {
        // Tambahkan ini:
        classpath 'com.google.gms:google-services:4.4.0'
    }
}
```

#### 9.2 Update `android/app/build.gradle`

```gradle
// Di paling bawah file, tambahkan:
apply plugin: 'com.google.gms.google-services'

// Pastikan minSdkVersion >= 21
android {
    defaultConfig {
        minSdkVersion 21  // atau lebih tinggi
        targetSdkVersion 34
    }
}
```

#### 9.3 Update `android/app/src/main/AndroidManifest.xml`

```xml
<manifest xmlns:android="http://schemas.android.com/apk/res/android">
    
    <!-- Permissions for notifications -->
    <uses-permission android:name="android.permission.INTERNET"/>
    <uses-permission android:name="android.permission.RECEIVE_BOOT_COMPLETED"/>
    <uses-permission android:name="android.permission.VIBRATE"/>
    <uses-permission android:name="android.permission.WAKE_LOCK"/>
    <uses-permission android:name="android.permission.POST_NOTIFICATIONS"/>
    
    <application
        android:label="Study Buddy"
        android:name="${applicationName}"
        android:icon="@mipmap/ic_launcher">
        
        <!-- Firebase notification service -->
        <service
            android:name="com.google.firebase.messaging.FirebaseMessagingService"
            android:exported="false">
            <intent-filter android:priority="1">
                <action android:name="com.google.firebase.MESSAGING_EVENT"/>
            </intent-filter>
        </service>
        
        <!-- Notification channel metadata -->
        <meta-data
            android:name="com.google.firebase.messaging.default_notification_channel_id"
            android:value="studybuddy_channel"/>
        
        <!-- ... rest of your application config ... -->
        
    </application>
</manifest>
```

### Step 10: Test Firebase Setup

#### 10.1 Test Firebase Initialization

Buat file test: `lib/firebase_test.dart`

```dart
import 'package:firebase_core/firebase_core.dart';
import 'package:firebase_messaging/firebase_messaging.dart';
import 'package:flutter/foundation.dart';

Future<void> testFirebaseSetup() async {
  try {
    // Test 1: Firebase initialized
    debugPrint('✅ Firebase initialized');

    // Test 2: Get FCM Token
    final messaging = FirebaseMessaging.instance;
    final token = await messaging.getToken();
    debugPrint('🔑 FCM Token: $token');

    // Test 3: Request permissions
    final settings = await messaging.requestPermission(
      alert: true,
      badge: true,
      sound: true,
    );

    debugPrint('🔔 Notification permission: ${settings.authorizationStatus}');

    // Test 4: Subscribe to topic
    await messaging.subscribeToTopic('all_users');
    debugPrint('📢 Subscribed to topic: all_users');

    debugPrint('🎉 Firebase setup complete!');
  } catch (e) {
    debugPrint('❌ Firebase test failed: $e');
  }
}
```

Panggil di `main.dart` setelah Firebase init:

```dart
await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);

// Test Firebase
await testFirebaseSetup();
```

## 🐛 Troubleshooting

### Error: "FirebaseException: No Firebase App"

**Solusi:**
```dart
// Pastikan Firebase.initializeApp() dipanggil SEBELUM runApp()
void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);
  runApp(MyApp());
}
```

### Error: "google-services.json missing"

**Solusi:**
1. Pastikan file ada di: `android/app/google-services.json`
2. Clean & rebuild:
   ```bash
   cd android
   ./gradlew clean
   cd ..
   flutter clean
   flutter pub get
   flutter run
   ```

### Error: "FCM token null"

**Solusi:**
1. Pastikan device punya Google Play Services
2. Di emulator: Gunakan emulator dengan Google Play
3. Request permission dulu:
   ```dart
   await FirebaseMessaging.instance.requestPermission();
   ```

### Error: "flutterfire command not found"

**Solusi:**
```bash
# Install ulang
dart pub global activate flutterfire_cli

# Windows - tambahkan ke PATH
$env:PATH += ";$env:LOCALAPPDATA\Pub\Cache\bin"

# Test
flutterfire --version
```
